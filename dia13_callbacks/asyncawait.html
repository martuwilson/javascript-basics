Async y await son una forma de trabajar con promesas de manera mas sencilla. Async es una palabra clave que se usa para declarar una funcion asincrona. Await es una palabra clave que se usa para esperar a que una promesa se resuelva. Await solo se puede usar dentro de una funcion async. Cuando se usa await una funcion async se detiene en esa linea hasta que la promesa se resuelva.
